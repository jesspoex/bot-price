# Token Price Bot CoinGecko

## by Jesspoex

__Steps to set up:__

1. Perbarui detail masukan untuk token yang diinginkan.
2. Tambahkan variabel lingkungan /(ENV) BOT_TOKEN dan GUILD_ID di area rahasia (variabel lingkungan). Gunakan token aplikasi bot dan id server Anda sebagai nilainya di sini.
3. Undang aplikasi bot Anda ke server perselisihan Anda dan berikan izin untuk mengubah/memodifikasi nama panggilan.
